# ACGA Theme

My personal theme based on the CGA palette, with the colors modified so they
meet the WCAG contrast requirements.

## Rationale

- Normal colors are modified to be at least 4.5:1 contrast (AA) against the
background color.
- Intense or (Bright) colors are modified to be at least 7:1 contrast (AAA)
against the background color.

## NOTE

- Intense Black is only required to be 4:5 contrast
